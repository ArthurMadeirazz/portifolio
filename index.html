MINHAS ESPECIALIDADES.

HTML5

HTML é a base de todas as minhas páginas web. É com ele que estruturo o conteúdo de meus sites, criando a marcação semântica que garante acessibilidade e uma boa experiência de navegação.

CSS

CSS é a tecnologia que utilizo para estilizar minhas páginas web, garantindo que elas sejam visualmente atraentes e responsivas. Com CSS, consigo criar layouts modernos, aplicar animações e transições, e ajustar a aparência para diferentes dispositivos.

JavaScript

JavaScript é a linguagem de programação que utilizo para tornar minhas páginas web interativas e dinâmicas. Com ela posso criar animações, validar formulários e desenvolver aplicações web completas.
